Constraints Satisfaction Problem - XYZ

"Simple Constraints on 3 Variables"

Description

There are three integers X, Y, and Z defined from 0 to 10. This variables are subject to the following constraints:

    X < Y
    X + Y = Z

Our goal is to find the solution that would maximize or minimize the objective function represented by the following integer expression:

    COST=3XY - 4Z

You can turn-on/turn-off these constraints or add some more constraints from the given list and try to solve the modified problem.
