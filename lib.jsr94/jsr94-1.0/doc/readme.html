Introduction

Reference Implementation
The Reference Implementation (RI) for JSR-94 is implemented as a wrapper over the Jess rule engine. You will need to download a copy (free for evaluation) from the Sandia website (link below).
See License Reference Implementation for license information on the reference implementation.

Technology Compatibility Kit
To help implementers to build compatible implementations, and evaluators to test implementations, we are providing the Technology Compatibility Kit (TCK) free of charge (with full source code). The TCK is built using the JUnit test framework.
See Technology Compatibility Kit for license information on the technology compatibility kit.

Installation
The downloaded distribution contains everything you need to review the specification, read the API JavaDoc, run the TCK and run the RI against a simple J2SE example. The Expert Group hopes to add more samples over time.

The top-level directory structure of the distribution is:

  • \ant: Jakarta ANT 1.5, used to build the specification, generate documentation and deploy samples.
  • \doc: Specification text and API documentation.
  • \lib: Compiled JARs of the specification API, reference implementation and samples.
  • \reports: Output reports from running the TCK against the RI.
  • \src: Java code for the TCK.

    • Running the Samples
    To run the samples based on the RI you will need to download an evaluation copy of Jess from Sandia National Laboratories. You should copy your downloaded jess.jar into the /lib directory.

    To run
    Goto the "lib" directory.
    cd lib
    Type the following command:
    java -jar jsr94-example.jar

    To view the source
    You can view the source in the src directory or click on the following link:
    org.jcp.jsr94.samples.Example.java
