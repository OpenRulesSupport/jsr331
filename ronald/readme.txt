Requirements
------------

Java and Groovy

If you want to plot charts then:

1. JFreeChart (put the jcommon and jfreechart on jars on the jar directory)

2. Builder wrapper from Dave Cherry
    http://www.thecoderscorner.com/blogEntry/show/16
    put the com directory on the archive on the lib directory

the jars on the jar directory plus the lib directory have to be on the
classpath
